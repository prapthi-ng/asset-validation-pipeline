# Asset Validation Pipeline (Python)

This project is a beginner-friendly Python script that simulates an animation pipeline asset validation system.

## Features
- Validates file types (.png, .psd, .jpg)
- Warns about unclean asset names (spaces or uppercase)
- Checks for missing size information
- Classifies assets as HEAVY or LIGHT based on size

## Skills Used
- Python
- Lists and Dictionaries
- Loops and Conditionals
- Functions
- Basic pipeline logic
